# Tips on the installation of the necessary libraries and generating data from Koch's model as descibed in the paper. 

The code comes from this reprository: 
https://github.com/j-koch/RDE1D and has been adapted to generate simulations with three co-rotating waves. 

After installing clawpack, to run the code properly, please substitute the file in /clawpack/riemann/euler_1D_py.py with the one euler_1D_py.py in this directory.
